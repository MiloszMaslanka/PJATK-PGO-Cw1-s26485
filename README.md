**## Etapy ćwiczenia**

**1. Inicjalizacja repozytorium lokalnego (git init)**

**2. Dodanie plików i commit (git add, git commit)**

**3. Dodanie zdalnego repo (git remote add origin)**

**4. Push na GitHub (git push -u origin main)**

**5. Praca na gałęziach feature/\***

**6. Merge do main**

**- git init**

**- git status**

**- git commit**

**- git branch -M main**

**- git checkout -b feature/...**

**- git merge**

**- git remote add origin**

**- git push -u origin <branch>**
